# Vocal Insight

An AI-powered speech analysis web application that helps users identify and improve communication issues in their speech. Submit a recording and receive detailed feedback on emotional tone, speech disorders, and bad habits — backed by machine learning models and a RAG-based coaching system.

---

## What It Analyses

| Category | What It Detects |
|----------|----------------|
| **Emotion** | Stress/anxiety tone, negative emotional dominance, flat/low expressiveness, emotion inconsistency |
| **Speech Disorders** | Stuttering, slurring (dysarthria) |
| **Bad Habits** | Filler words, pace instability |

Each detected issue is paired with personalised, context-aware coaching feedback generated by an LLM using a RAG knowledge base.

---

## System Architecture

The application runs as 6 independent services:

| Service | Stack | Port |
|---------|-------|------|
| Frontend | React + Vite | 5173 |
| Auth Backend | Node.js + Express | 3001 |
| Emotion Recognition API | Python + FastAPI + TensorFlow | 8000 |
| Disorder Detection API | Python + FastAPI + PyTorch | 5000 |
| Bad Habit Detection API | Python + FastAPI + PyTorch | 8001 |
| Feedback / RAG API | Python + FastAPI + FAISS + OpenRouter | 5001 |

## Prerequisites

Make sure the following are installed on your machine before proceeding:

- [Python 3.10+](https://www.python.org/downloads/)
- [Node.js 18+](https://nodejs.org/)
- An [OpenRouter](https://openrouter.ai/keys) API key (Copy to Clipboard)

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/chanukaWijeratna/DSGP_Group_18.git
cd DSGP_Group_18
```

### 2. Download the model files

Download the `model-config` folder from the shared Google Drive link and place it inside the repository root so the structure looks like this:

Folder Link - https://drive.google.com/file/d/1vbBvoHLZVefpuotv32z70MGEjgdfSEBi/view?usp=sharing

```
DSGP_Group_18/
├── model-config/
│   ├── bad-habit/models/
│   ├── disorder/models/
│   └── emotion/models/
├── backend/
├── frontend/
├── package.json
├── requirements.txt
├── setup.bat
├── start.bat
└── README.md
```

### 3. Run setup

Double-click `setup.bat` or run it from a terminal:

```bash
setup.bat
```

The script will automatically:
- Copy ML models to the correct backend folders
- Create a Python virtual environment
- Prompt you to **paste your OpenRouter API key** — have it copied to your clipboard
- Install all Python and Node.js dependencies
- Generate the RAG knowledge index

### 4. Launch the application

Once setup completes, double-click `start.bat` or run:

```bash
start.bat
```

This opens all 6 services in separate terminal windows. Once they are all running, open your browser and go to:

```
http://localhost:5173
```

## Usage

1. **Sign up** for an account or log in
2. On the **Record** page, record your speech directly or upload an audio file
3. Optionally describe the speaking scenario (e.g. "job interview", "class presentation")
4. Click **Submit for Analysis**
5. View your results — emotion breakdown, disorder detection, habit analysis, and personalised coaching feedback
6. Track your progress over time through the history section

> **Note:** Audio upload is supported but not recommended — live recordings produce more accurate results.
